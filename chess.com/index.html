La commande help

La commande (sous sa forme complète : !match__help) est la commande faisait un cours résumé des syntaxes des commandes du bot.

La commande register

La commande (sous sa forme complète : !match_register) est la commande servant à s'enregister auprès du bot en y saisissant les paramètres associés,
il faut donc joindre à la commande les paramètres dans l'ordre suivant : @Joueur Classe Nom_joueur.

@Joueur = ping discord de vous même
Classe = classe dans laquelle vous allez être enregistré
Nom_joueur = votre prénom

La commande result

La commande (sous sa forme complète : !match_result) est la commande servant à fournir un résultat de match qu'il faudra confirmer par la suite,
la commande s'éffectue donc accompagné des paramètres : ScoreJ1 ClasseJ2 NomJ2 ScoreJ2.

ScoreJ1 = score de la personne associé à @Joueur1
ClasseJ2 = classe de la personne associe à @Joueur2
NomJ2 = nom enregistré de la personne @Joueur2
ScoreJ2 = score de la personne associé à @Joueur2

La commande result_confirm

La commande (sous sa forme complète : !match_result_confirm) est donc la commande permettant à un joueur A ayant joué une partie avec un autre joueur B qui a donné le résultat, de confirmer le résultat donné par le joueur B.
Pour éviter tout soupçons de triche dans les paramètres nous allons retrouver : @Toooom#2689 @JoueurB *arg.
A savoir que le paramètre "*arg" correspond à une lettre entre 'y' et 'n' correspondant respectivement à "yes" pour confirmer et "no" pour refuser le résultat donné auparavant.

@Toooom#2689 = ping discord de l'administrateur Tom
@JoueurB = ping discord du joueur ayant fait la commande result pour le résultat de votre match
arg = caractère choisit entre 'y' pour confirmer les résultats donnés auparavant et 'n' pour refuser les résultats donnés

La commande score_class

La commande (sous sa forme complète : !match_score_class) permet d'obtenir le score total d'une classe donné; voici les paramètres de la commande : Classe.

Classe = classe dont vous voulez obtenir le score

La commande score_player

La commande (sous sa forme complète : !match_score_player) permet donc d'obtenir le score total d'un joueur d'une certaine classe, la liste des paramètre est : Classe Nom_joueur.

Classe = classe du joueur concerné
Nom_joueur = joueur dont vous voulez obtenir le score

La commande modif_class

La commande (sous sa forme complète : !match_modif_class) grâce à cette commande vous allez pouvoir changer votre classe par une nouvelle, les paramètres seront donc : Classe.

Classe = votre future classe

La commande modif_player

La commande (sous sa forme complète : !match_modif_player) va donc permettrre un changement de nom, il faudra donc fournir en paramètre : Classe Nom1 Nom2.

Classe = classe dans laquelle vous êtes
Nom1 = votre nom actuel
Nom2 = votre futur nom

La commande edit_color_profil

La commande (sous sa forme complète : !match_edit_color_profil) va donc permettrre un changement de la couleur lors de l'affichage du profil de la personne via la commande qui va suivre celle-ci, il faudra donc fournir en paramètre : Hexa.

Hexa = code héxadécimal de la couleur choisie (ex : rouge => ff0000, le signe '#' ou '0x' ne doit pas être présent devant le code héxadécimal)

La commande show_profil

La commande (sous sa forme complète : !match_show_profil) va donc permettrre d'aficher son profil, cette commande ne requiert d'aucun paramètre
