La commande register

La commande est la commande servant à s'enregister auprès du bot en y saisissant les paramètres associés, la commande accompagnée de paramètre ressemblera donc à ça :
!register @Joueur Classe Nom_joueur.

@Joueur = ping discord de vous même
Classe = classe dans laquelle vous allez être enregistré
Nom_joueur = votre prénom

La commande result

La commande est la commande servant à fournir un résultat de match qu'il faudra confirmer par la suite, elle prendra la forme de :
!result ScoreJ1 ClasseJ2 NomJ2 ScoreJ2.

ScoreJ1 = votre score lors de la partie
ClasseJ2 = classe de l'adversaire
NomJ2 = nom enregistré de l'adversaire
ScoreJ2 = score de l'adversaire

La commande result_confirm

La commande sert à confirmer le résultat d'un match vous concernant déposé auparavant, la commande prendra donc la forme suivant :
!result_confirm @Toooom#2689 @JoueurB arg

@Toooom#2689 = ping discord de l'administrateur Tom
@JoueurB = ping discord du joueur ayant fait la commande result pour votre match
arg = caractère choisit entre 'y' pour confirmer les résultats donnés auparavant et 'n' pour refuser les résultats donnés

La commande score_class

La commande permet d'obtenir le score total d'une classe donné, la commade est :
!score_class Classe.

Classe = classe dont vous voulez obtenir le score

La commande edit_color_profil

La commande va donc permettrre un changement de la couleur lors de l'affichage du profil de la personne via la commande suivant :
!edit_color_profil Hexa.

Hexa = code héxadécimal de la couleur choisie (ex : rouge => ff0000, le signe '#' ou '0x' ne doit pas être présent devant le code héxadécimal)

La commande show_profil

La commande va donc permettrre d'aficher son profil, un paramètre peut-être rajouter (ou non) si l'on veux présicier le profil de qu'elle personne nous voulons cette commande s'exécute simplement en faisant :
!show_profil @Joueur.

@Joueur = ping discord de la personne dont vous voulez le profil (si vous voulez votre profil pas besoin de rajouter ce paramètre)

La commande answer_sondage

Cette commande vous permettra de donner votre avis à un sondage créé via son appel qui est le suivant :
!answer_sondage Id Choice.

Id = id du sondage (demandez à un administrateur si besoin)
Choice = votre choix parmi les 2 choix (si les deux choix sont des suites de mots veuillez renseigner votre choix entre guillements : !answer_sondage 1 "mon choix")
